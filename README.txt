In order to change the simulated scenario edit file "/src/D2D-Sim.cpp" and recompile.
In order to run the simulation execute file "/Release/D2D-Sim.exe" or "/Release/D2D-Sim.exe [seed_for_srand()]".

Code developed using Microsoft Visual Studio 2010.

Original LTE-Sim:
G. Piro, L. Grieco, G. Boggia and F. Capozzi, "Simulating LTE Cellular Systems: An Open-Source Framework," Vehicular Technology, IEEE Transactions on, vol. 60, no. 2, pp. 498-513, 2011.
